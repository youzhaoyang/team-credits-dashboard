# 团队积分消耗看板

一个自包含的单文件静态看板（HTML + Chart.js），展示团队按**时间 / 模型 / 成员**三个维度的积分消耗，并提供消耗明细的服务端分页演示、CSV 导出等交互。

> 数据为前端确定性生成的**示例数据**（页面右上角标注「示例数据」），三个视图同源对账。

## 在线预览

通过 GitHub Pages 部署：见仓库 **Settings → Pages** 给出的地址。

## 本地查看

直接用浏览器打开 `index.html` 即可，无需构建。Chart.js 通过 CDN 引入。

## 结构

- `index.html` — 全部页面、样式与逻辑（无构建步骤、无依赖安装）
